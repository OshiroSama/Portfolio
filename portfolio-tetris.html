Personal Project Tetris

Final project for my Fullstack bootcamp

The idea came to me as I was thinking about fun games with a complex mechanism that would be fun to make. I created a postgreSQL database to contain users and their scores, while the frontend was made with react JS.

The challenging part of this project was to understand how to draw the board and add score everytime a row has been filled. It wasn't long before I began playing with it and changing values according to what I see fit and customizing it to my own preferences

This is one project I am very proud to present. I had a great time working on the algorithms and designing the visuals
